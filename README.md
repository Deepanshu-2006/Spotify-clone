# Spotify Clone

A responsive **Spotify-inspired music streaming UI** built with **HTML, CSS, and JavaScript**.  
This project focuses on recreating the look-and-feel of Spotify (player controls, layout, playlists, etc.) for learning and practice.

---

## Demo

- Run locally by opening `index.html` in your browser (see **Getting Started**).

---

## Features

- Spotify-like modern UI
- Responsive layout (works on desktop and mobile)
- Music player controls (play/pause, next/previous, seek)
- Playlist / track list view
- Progress bar + current time / duration (if implemented in your JS)
- Volume control (if implemented)

> Note: Feature availability depends on what’s implemented in your current codebase. Update this list to match your project.

---

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

## Getting Started

### 1) Clone the repository
```bash
git clone https://github.com/Deepanshu-2006/Spotify-clone.git
```

### 2) Open the project
```bash
cd Spotify-clone
```

### 3) Run
- If it’s a static project: open `index.html` directly in your browser  
  **or**
- Use a simple local server (recommended):

```bash
# Python 3
python -m http.server 5500
```

Then open:
- `http://localhost:5500`
